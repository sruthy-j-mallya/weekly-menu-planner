# Weekly Menu Planner

A simple menu planner where you create a list of dishes with name, category and color, and then drag and drop these dishes to the menu table.

# Tech Stack
- HTML
- CSS
- JavaScript
- No, database, stores the data in localStorage.

Created with [a simple prompt using Claude](https://claude.ai/share/093368d7-c471-41dc-a7a9-8afbe62cf1e3).
